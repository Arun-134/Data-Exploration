# Data-Exploration
Focuses on Excel-based data exploration and analysis using a structured product dataset. The goal is to build foundational skills in data summarization, logical functions, conditional aggregation, and text manipulation—all essential techniques for preparing data for deeper analytics.
# 📊 Project Overview
This project is part of the Data Analytics (DA) Module – Excel, focusing on foundational data exploration techniques using Microsoft Excel.
The dataset contains product information such as Product ID, Product Name, Brand, Category, Quantity, and Price.
The goal of this assignment is to apply Excel formulas to summarize data, categorize values, extract text, and perform conditional calculations.
# 🧠 Skills Demonstrated
Data Summarization (SUM, COUNT, AVERAGE)

Statistical Analysis (MIN, MAX)

Logical Functions (IF, IFS)

Conditional Aggregation (SUMIF, COUNTIF)

Text Manipulation (LEFT, RIGHT, MID)

Spreadsheet Structuring & Formula Application
